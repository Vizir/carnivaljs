# Carnival JS

## Installation

Install dependencies


    $ npm install


Install client dependencies


    $ bower install


## Run the project


Run the command

  $ npm run start:watch


Open the following url

    http://localhost:3010/

## Grunt tasks

run server and file watcher


    $ npm run start:watch


run tests


    $ npm test:watch


create carnival.js and carnival.min.js on 'dist' folder


    $ npm run build
